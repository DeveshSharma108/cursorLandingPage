# Cursor Landing Page Clone

A desktop-first recreation of the **Cursor developer tool landing page**, built using only **HTML and CSS** as part of the Web Dev Cohort assignment.

The goal of this project was **visual accuracy** matching the layout, colors, typography, and structure of the original Cursor website.

Original site: [https://cursor.com](https://cursor.com)
